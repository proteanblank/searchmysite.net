Spacers

Spacers work like this: sms-[side]-4, where edge is top or bottom for now. Top would be sms-t-20. Bottom would be sms-b-16. We're using pixel values to have a reference designers and developers can easily understand, but the value is actually a REM value. It's just that REMS are not the most user friendly.

We don't have all the values with 4 pixel or 8 pixel increments, its mostly using an 8px increment rule, except for some lower numbers. We've added only what we've needed, so you won't find every single 8 pixel increment up to 80. Feel free to add new values as needed, but stick to the 8 pixel increment rule.
